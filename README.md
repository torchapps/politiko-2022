# [Politi-ko 2022](https://torchapps.github.io/politiko-2022/)

Match your political views with those of your political candidates!

## Requirements

* `python3`

## Data setup

1. Update the `*.csv` files
2. Run `make data.js`

## Dev setup

`make dev`

## Deployment

Pushing to `main` automatically redeploys the GitHub page.
